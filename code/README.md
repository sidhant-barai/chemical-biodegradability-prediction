# Source Code

This folder contains the primary Python script used for the machine learning pipeline.

### 📁 Files
- `dm230118267.py`: Master script including:
  - **Data Preprocessing:** Outlier removal using Z-score.
  - **Exploratory Data Analysis:** Correlation heatmaps and box plots.
  - **Dimensionality Reduction:** PCA implementation (reducing 41 features to 26).
  - **Model Training:** Random Forest Classifier & Regressor implementations.
  - **Validation:** 5-fold cross-validation and MSE tracking.

### ⚙️ Requirements
To run this script, you will need:
- `pandas`, `seaborn`, `matplotlib`, `scikit-learn`
